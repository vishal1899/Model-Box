What is Model Box?

A modal is a dialog box/popup window that is displayed on top of the current page.

Modals provide a nice way to provide extra details without navigating away from the page completely, but they hide your current context and feel a bit 90's. Popovers are essentially stylish tooltips which provide a little extra information without losing context.
